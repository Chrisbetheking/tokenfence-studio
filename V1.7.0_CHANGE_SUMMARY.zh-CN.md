# TokenFence Studio v1.7.0 改造摘要

这次不是给原来的 DeepSeek 页面继续堆按钮，而是把产品重新整理为一个可持续扩展的 **Safe AI Workspace**。

## 已完成

- 13 种 Provider 模板与多个独立 Profile；
- 修复保存 DeepSeek 后工作台自动切换到 Local Sandbox 的问题；
- 密钥由 Rust 后端按 Profile 直接读取 Keychain / Credential Manager，不返回 WebView；
- 本地 Token 估算、保守压缩和平衡压缩；
- TXT、代码、PDF、DOCX、XLSX/XLS、图片 OCR 本地处理；
- 按代码、PDF、OCR、表格和通用任务配置模型路由；
- 12 个内置 Skills 与 3 个默认 Agent；
- GitHub Release 更新检查与安装包展示；
- 现代化 macOS Overlay 标题栏与全新导航；
- Apple Silicon / Intel 双架构构建及 Release 工作流；
- 隐私测试、依赖审计和完整性校验。

## Computer Use 边界

v1.7.0 已提供权限模型、能力检测、Computer Use Guard 与界面入口，但屏幕捕获、鼠标、键盘、项目写入和终端操作仍为 Planned。后续会采用逐步授权、范围限制、审计回执和紧急停止，而不是增加任意命令后门。

## 下一阶段

优先推进 GitHub 仓库连接、补丁式代码编辑、受限命令执行、Diff 审查、测试与回滚，然后再进入受控 Computer Use 和 MCP / Skills 扩展生态。
