# Chris Studio v2.0.0 改造摘要

本次版本不再只增加展示页面，而是把此前缺失的核心能力接入实际链路：

1. **Coding Agent**：选择受限仓库目录，AI 基于脱敏仓库树、Git 状态、当前 Diff 和选中文件生成计划与统一 Diff；用户审查后才能检查和应用。
2. **GitHub**：PAT 存入系统凭证库，支持仓库与 Issue 读取、分支、提交、推送和创建 Pull Request。
3. **Computer Use**：macOS 截图、坐标点击、文本输入、白名单按键和审计记录，每个动作单独确认。
4. **Skills/MCP**：20 个内置 Skills、自定义 Skill 编辑、导入导出、MCP JSON-RPC 工具连接和逐次授权。
5. **文档智能**：扫描 PDF OCR、中文/中英 OCR、本地知识分块与检索、真实视觉图片附件。
6. **Token 控制**：单次上限、每日预算、压缩与用量统计。
7. **发布**：Apple Silicon/Intel 双架构，支持 Developer ID 签名与公证；无凭证时提供社区安装助手。

保留的限制均为明确安全边界：不提供任意 Shell、不允许模型自行批准写入、不允许无人值守 Computer Use、不静默安装更新。
