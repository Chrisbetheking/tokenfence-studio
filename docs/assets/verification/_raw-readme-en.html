24## Overview
25
26TokenFence Studio is a local-first AI workspace that sits between users and large language models.
27
28It is not just another chat UI. It builds a pre-LLM layer that can inspect, clean, protect, chunk, and route user input before it reaches a model.
29
30## Feature Matrix
31
32| Area | Capability | Status |
33|---|---|---|
34| Prompt Guard | Detect secrets, credentials, tokens, emails, phone numbers, database URLs, and risky prompt content | Working |

36| Document Intelligence | PDF / DOCX / image OCR parsing, cleaning, and chunking | Working / experimental depending on input |

41| Local Runtime | Execute approved local tasks and save logs | Verified |
42| Obsidian Memory | Write output notes into a test vault | Verified |

64## Known Limitations
65
66- This is a release candidate, not the final v1.0 production release.
67- Windows Desktop is an unsigned i686 experimental build.
68- Windows x64 is blocked by missing MSVC linker / 64-bit MinGW-w64 in the current build environment.
69- macOS artifact is CI-prepared but not yet verified.
70- Android is Mobile Lite, not full desktop parity.
71- Play Store production signing is not available.
72- Provider calls require user-provided API keys.
73- Computer Use full control remains experimental.
