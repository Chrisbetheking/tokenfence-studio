## Current Project Status

TokenFence Studio is currently an early-stage open-source prototype.

The project is focused on building a local-first pre-LLM layer for:

- Prompt safety
- Sensitive data detection
- Redaction
- Intent routing
- Context compression
- Multi-provider routing
- Multi-model comparison
- File-level model routing
- Agent-ready workflows

Feedback, suggestions, issues, and contributions are very welcome.
