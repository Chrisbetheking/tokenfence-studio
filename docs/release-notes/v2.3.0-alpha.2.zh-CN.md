# Chris Studio v2.3.0-alpha.2

本次 Alpha 仅修复多模型 Agent 发送体验，不修改已通过验收的 Planner、Executor、Reviewer、Computer Use 和可靠执行中心逻辑。

## 修复

- 请求通过安全审查、Provider、Token 预算等发送前门禁并写入会话后，输入框立即清空。
- Planner → Executor → Reviewer 运行期间不再持续显示已经发送的旧提示词。
- 输入框清空后自动恢复焦点，便于继续输入下一条消息。
- Provider 未连接、Token 超限、用户拒绝预算确认等“尚未真正发送”的情况继续保留草稿，避免误丢输入。

## 版本

- `v2.3.0-alpha.2`
- 仍应作为 Pre-release 发布。
- `v2.2.0` 继续保持稳定 Latest。
