# TokenFence Studio v0.3.0

This release adds the first **Document Intelligence Pipeline** prototype.

## Highlights

- New Docs workspace tab.
- Document upload and pasted document processing.
- Lightweight parsing for PDF, DOCX, text, Markdown, logs, JSON, and code files.
- Image placeholder flow for future OCR / vision model integration.
- Cleaning pipeline for page numbers, repeated headers/footers, blank lines, and duplicated paragraphs.
- Sensitive data scanning on cleaned document content.
- RAG-ready `chunks.json` generation.
- Clean Markdown export.
- File-level suggested model routing.
- Example assets under `examples/document-intelligence/`.

## Status

This is still a prototype. The parsing layer is intentionally lightweight and will need stronger extractors for production usage.

Future work:

- Better PDF layout extraction.
- OCR / vision model adapters.
- Search grounding with citations.
- Judge model for multi-model result merging.
- Cost and latency-aware model routing.
